# 30 Demo and Operator Polish

## Goal

Make the project easy to run, explain, and evaluate.

## Scope

In scope:

- README update to match actual commands.
- Quickstart using venv and `PYTHONPATH` fallback.
- Demo script.
- Troubleshooting guide for Homebrew Python, SSL, and editable installs.
- Architecture diagram updates.
- Interview explanation guide.
- Seed examples and expected screenshots/output snippets.

Out of scope:

- New core workflow behavior.
- Production deployment.

## Tests

- README quickstart commands are smoke-tested where possible.
- Demo script runs fake pipeline end to end.
- Docs link to existing module and learn documents.

## Done Criteria

- A fresh reviewer can clone, run, inspect, and understand the project in under
  10 minutes.
